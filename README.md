# Deutsch-Jozsa Algorithm - Emerging Technologies

## Overview

This repository implements the Deutsch-Jozsa quantum algorithm using Qiskit, 
exploring the transition from classical to quantum computing through five problems.

## Problems

- **Problem 1:** Random Boolean function generator (constant and balanced functions)
- **Problem 2:** Classical algorithm for determining function type
- **Problem 3:** Quantum oracle implementations
- **Problem 4:** Deutsch's algorithm for single-input functions
- **Problem 5:** Full Deutsch-Jozsa algorithm for 4-input functions

## Setup

1. Clone the repository
2. Install dependencies:
```bash
pip install -r requirements.txt
```
3. Open the notebook:
```bash
jupyter lab
```
4. Run all cells in order (Kernel → Restart Kernel and Run All Cells)

All dependencies are listed in `requirements.txt`. Key libraries:
